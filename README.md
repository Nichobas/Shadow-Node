## Français

**Shadow Node** est un jeu de puzzle narratif sur le thème du hacking, conçu pour iPhone. Tu enquêtes (OSINT), tu craftes des exploits, tu négocies sur le darknet et tu prends des décisions dans des conversations IRC. Aucune saisie clavier, tout se joue au tap.

### Configuration requise

| | |
|---|---|
| Appareil | iPhone uniquement |
| iOS | 18.0 ou plus récent |
| Orientation | Paysage |
| Connexion | Hors-ligne (sauf Game Center) |
| Langues | Français, Anglais |
| Achats intégrés | Aucun |
| Espace requis | ~50 Mo |

### Comment jouer

- Toutes les interactions se font au **tap** : pas de clavier, pas de saisie texte.
- Tiens ton iPhone en **mode paysage**.
- Suis le **journal de mission** pour identifier ton prochain objectif.
- Le **dock** en bas d'écran débloque progressivement de nouveaux outils selon ton avancée (terminal, OSINT, darknet, chat).

### Questions fréquentes

**Le jeu reste sur fond noir au lancement.**
Force la fermeture (swipe vers le haut depuis l'écran d'accueil), puis relance. Si le problème persiste, vérifie qu'iOS est à jour (≥ 18.0).

**Je suis bloqué dans une mission.**
Chaque mission contient un système d'aide intégré. Touche l'icône d'aide en haut de l'écran de chat ou retourne sur le tableau d'enquête (*evidence board*) pour reconstituer les indices.

**J'ai perdu ma progression.**
La progression est sauvegardée localement sur ton appareil. Une réinstallation efface la sauvegarde — pense à activer iCloud Backup pour ton appareil afin de protéger tes données.

**Game Center ne se connecte pas.**
Va dans *Réglages iOS → Game Center* et vérifie que tu es connecté. Relance ensuite Shadow Node. Game Center est optionnel : tu peux jouer sans.

**Le son ne fonctionne pas.**
Vérifie que le mode silencieux n'est pas activé sur ton iPhone et que le volume média est monté. Tu peux aussi activer ou couper le son depuis les réglages dans le menu du jeu.

**Comment changer la langue ?**
Shadow Node suit la langue de ton iPhone. Pour changer : *Réglages iOS → Général → Langue et région*, ou *Réglages iOS → Shadow Node → Langue préférée*.

**Comment supprimer ma sauvegarde et recommencer ?**
Désinstalle puis réinstalle l'application. *Attention : cette action est définitive.*

**Le jeu est-il jouable hors-ligne ?**
Oui, entièrement. Aucune connexion Internet n'est requise pour jouer. Game Center et l'App Store nécessitent une connexion uniquement pour les fonctions associées.

### Confidentialité

Shadow Node **ne collecte aucune donnée personnelle**. Aucun tracking, aucune analytics tierce, aucun identifiant publicitaire. Toute ta progression reste sur ton appareil. La seule donnée éventuellement transmise concerne Game Center (succès et identifiant de joueur Apple), uniquement si tu choisis de l'activer.

## English

**Shadow Node** is a narrative hacking puzzle game for iPhone. Investigate (OSINT), craft exploits, deal on the darknet, and make choices in IRC-style conversations. No typing — everything plays through taps.

### Requirements

| | |
|---|---|
| Device | iPhone only |
| iOS | 18.0 or newer |
| Orientation | Landscape |
| Connection | Offline (except Game Center) |
| Languages | English, French |
| In-app purchases | None |
| Storage | ~50 MB |

### How to play

- Every interaction is a **tap** — no keyboard, no text input.
- Hold your iPhone in **landscape mode**.
- Follow the **mission log** to identify your next objective.
- The **dock** at the bottom unlocks new tools (terminal, OSINT, darknet, chat) as you progress.

### Frequently asked questions

**The game stays on a black screen at launch.**
Force-quit (swipe up from the home screen), then relaunch. If the issue persists, make sure iOS is up to date (≥ 18.0).

**I'm stuck in a mission.**
Each mission has built-in hints. Tap the help icon at the top of the chat screen, or return to the evidence board to reconnect clues.

**I lost my progress.**
Progress is saved locally on your device. Reinstalling the app clears your save — enable iCloud Backup on your device to protect your data.

**Game Center won't connect.**
Open *iOS Settings → Game Center* and confirm you're signed in, then relaunch Shadow Node. Game Center is optional — the game is fully playable without it.

**No sound.**
Make sure silent mode is off and media volume is up. You can also toggle sound from the in-game settings menu.

**How do I change the language?**
Shadow Node follows your iPhone's language. Change it via *iOS Settings → General → Language & Region*, or *iOS Settings → Shadow Node → Preferred Language*.

**How do I delete my save and start over?**
Uninstall and reinstall the app. *Warning: this action is permanent.*

**Can I play offline?**
Yes, entirely. No internet connection is needed. Game Center and the App Store only require a connection for their respective features.

### Privacy

Shadow Node **collects no personal data**. No tracking, no third-party analytics, no advertising identifiers. All your progress stays on your device. The only data possibly transmitted relates to Game Center (achievements and Apple player ID), and only if you choose to enable it.
